## Invoices-Stripe App

Апликација за креирање, управување и плаќање фактури со интеграција со Stripe.

## Опис

Оваа апликација овозможува управување со клиенти и фактури, како и безбедно онлајн плаќање преку Stripe.

Корисникот може да креира фактура, да внесе податоци за клиентот и износот за наплата, по што автоматски се генерира линк за плаќање. Линкот се испраќа по email до клиентот, кој може да ја плати фактурата преку Stripe Checkout.

По успешно или неуспешно плаќање, статусот на фактурата автоматски се ажурира во системот.

##  Користени технологии

* Java 21
* Spring Boot
* Spring Data JPA
* PostgreSQL
* Stripe API
* Spring Mail
* Swagger / OpenAPI

## Функционалности

✅ Управување со клиенти

✅ Креирање фактури

✅ Генерирање линк за плаќање

✅ Испраќање email до клиент

✅ Интеграција со Stripe Checkout

✅ Следење на статус на плаќање

✅ Ажурирање на статус:

* PENDING
* PAID
* FAILED

##  Цел на проектот

Проектот е изработен со цел да демонстрира развој на full-stack веб апликација со интеграција на надворешен платежен систем, работа со база на податоци и испраќање email известувања.
